# chaos-sim 計画書

## 概要

ChaosTCG 一人回しシミュレーター。ws-simと同等の機能をChaos向けに実装する。

## テーマ

- カラー: **青と白**（ChaosTCGのブランドカラー）
- 背景: ダークネイビー（#1a2332）
- アクセント: ブルー（#4a9fd4）
- ボーダー/テキスト: 白系（#e8f0ff）
- ステージ枠: 青（#3a7bd5）

## フィールド構成

WSとの違いを反映したレイアウト:

```
【相手フィールド（上）】

  [前衛左]  [前衛右]
     [パートナー]
  [後衛左]  [後衛右]

  [デッキ] [控え室] [Exデッキ] [バックヤード]

─────────────────────────────────

【自分フィールド（下）】

  [手札]

  [後衛左]  [後衛右]
     [パートナー]
  [前衛左]  [前衛右]

  [デッキ] [控え室] [Exデッキ] [バックヤード]
```

### ゾーン一覧（プレイヤーごと）

| ゾーン | 説明 |
|--------|------|
| パートナーエリア | 中央1枠。パートナー専用 |
| 前衛エリア×2 | 攻撃参加可能なフレンド |
| 後衛エリア×2 | 後衛フレンド |
| メインデッキ | 20枚 |
| エクストラデッキ | エクストラカード置き場 |
| 控え室 | 墓地相当 |
| バックヤード | 特殊領域 |
| 手札 | 非公開 |
| 解決領域 | 一時領域 |

## WSとの主な差異

| 項目 | WS | Chaos |
|------|-----|-------|
| 舞台構成 | 前列3+後列3=6枠 | 前衛2+後衛2+パートナー1=5枠 |
| デッキ枚数 | 50枚 | 20枚（メイン）+ エクストラ |
| ダメージ | ソウル→クロック→レベル | バトル（ATK vs BP）→裏 |
| 勝利条件 | Lv4到達で敗北 | パートナーのBP0で敗北 |
| CX/トリガー | CXでキャンセル | なし |
| 特殊領域 | 思い出、クロック、レベル、ストック | バックヤード |
| カード状態 | スタンド/レスト/リバース | 表スタンド/表レスト/裏 |
